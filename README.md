#  AI Text Sentiment Analyzer

A simple Python-based AI project that analyzes the sentiment of user-input text and classifies it as **Positive**, **Negative**, or **Neutral** using Natural Language Processing (NLP).



##  Project Description

This project uses a pre-trained NLP model to understand human language and determine emotional sentiment from text. It is a beginner-friendly AI project that demonstrates how real-world applications use AI without training complex machine learning models.



##  Features

- Accepts text input from the user
- Analyzes sentiment using NLP
- Displays:
  - Sentiment label (Positive / Negative / Neutral)
  - Polarity score
  - Confidence percentage
- Clean and modular Python code


##  Technologies Used

- Python 3
- TextBlob (Natural Language Processing library)
